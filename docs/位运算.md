

## 位运算基本操作

| 按位与   | a&b  |
| -------- | ---- |
| 按位或   | a\|b |
| 按位异或 | a^b  |
| 按位取反 | ~a   |
| 左移     | a<<b |
| 右移     | a>>b |

 ## Tricks

```
n << 1  # 乘以2运算
n >> 1  # 除以2运算
n << m  # 乘以2的m次方
n >> m  # 除以2的m次方
(n & 1) == 1 # 判断一个数的奇偶性
x ^ y >= 0 # 判断两个数符号是否相同
2 << (n-1) # 计算2的n次方

# 不用临时变量交换两个数
a ^= b
b ^= a 
a ^= b
a = 10, b = 20    a = 01010   b = 10100
01010 ^= 10100 --> a = 11110
10100 ^= 11110 --> b = 01010
11110 ^= 01010 --> a = 10100
```
